01. Come funziona

Dalla chat al backlog ufficiale

Descrivi il lavoro in linguaggio naturale. L'agente verifica duplicati, crea l'issue con label corrette e referenzia il numero nei commit. GitHub Issues diventa la fonte di verità.

Parli in chat

«Traccia bug…», «Apri issue per…», «Elenca le issue aperte». Non serve ricordare la sintassi CLI.

L'agente esegue

La regola .cursor/rules/github-issues.mdc obbliga l'agente a lanciare lo script Python — non a chiederti comandi da incollare.

Issue su GitHub

Backlog condiviso, label, template web, chiusura automatica con Fixes #N nelle PR.
